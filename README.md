# Regresion lineal multiple

**Nombre:** Juan Rodrigo Guerrero De la Garza  
**Matrícula:** 611471  

**Objetivo:** Ser capaz de generar una regresión lineal múltiple y determinar si existe una asociación significativa lineal entre alguna variable del sistema y su salida.

---

En esta actividad se utilizó una base de datos de la NASA con la que trabajaron para tratar de determinar perfiles aerodinámicos ideales ante distintas condiciones, como: la velocidad del viento y ángulo de ataque del mismo.donde podrás encontrar información para
1,052 observaciones distintas, con 6 mediciones para cada una de ellas. Los datos se
descargaron del [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/291/airfoil+self+noise), y originalmente se publicaron en el [NASA Reference Publication 1218](https://ntrs.nasa.gov/api/citations/19890016302/downloads/19890016302.pdf).
